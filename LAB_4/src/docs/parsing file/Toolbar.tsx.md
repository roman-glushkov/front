# Разбор Toolbar.tsx

Этот компонент представляет панель инструментов редактора презентаций. Он содержит набор кнопок для выполнения различных действий: добавление, удаление, перемещение слайдов, а также редактирование элементов и изменение фона.

interface Props { ... } — описание пропсов:
onAction — функция, которая вызывается при нажатии на любую кнопку и получает строку с названием действия.

export default function Toolbar({ onAction }: Props) — определение функционального компонента с деструктуризацией пропсов.

return (...) — JSX-разметка панели инструментов.

Каждая кнопка при нажатии вызывает функцию onAction с соответствующим названием действия, чтобы обработчик из хука useEditor выполнил нужное изменение.
