# Лабораторная работа №4

## Changelog по сравнению с Лабораторной работой №3

### 1. Структура проекта

| Было (ЛР-3)                    | Стало (ЛР-4)                                             |
| ------------------------------ | -------------------------------------------------------- |
| Весь код в `App.tsx`           | Код разделён на модули                                   |
| Логика и интерфейс смешаны     | Вынесена логика в `editor.ts`, UI в отдельные компоненты |
| Один CSS-файл со всеми стилями | Стили разделены по компонентам + глобальные              |

### 2. Компоненты интерфейса и стилизация

| Компонент       | Было (ЛР-3)                             | Стало (ЛР-4)                                                         |
| --------------- | --------------------------------------- | -------------------------------------------------------------------- |
| **Toolbar**     | Кнопки действий в `App.tsx`             | Вынесен в `common/Toolbar.tsx`, стили в `common/Toolbar.css`         |
| **SlidesPanel** | Список слайдов встроен в `App.tsx`      | Вынесен в `common/SlidesPanel.tsx`, стили в `common/SlidesPanel.css` |
| **Workspace**   | Рабочая область слайда внутри `App.tsx` | Вынесен в `common/Workspace.tsx`, стили в `common/Workspace.css`     |

### 3. Логика (Editor)

| Было (ЛР-3)                                                    | Стало (ЛР-4)                                                       |
| -------------------------------------------------------------- | ------------------------------------------------------------------ |
| Состояние презентации напрямую через `useState`                | Создан класс `Editor` (`store/editor.ts`)                          |
| Все действия (добавить/удалить/переместить) — внутри `App.tsx` | Действия реализованы методами `Editor`                             |
| Логика и UI тесно связаны                                      | Разделение: `App.tsx` вызывает методы `Editor`, UI остаётся чистым |
| Трудно расширять                                               | Удобно добавлять новые методы и экшены                             |

### 4. Стилизация

| Было (ЛР-3)                             | Стало (ЛР-4)                                                                                    |
| --------------------------------------- | ----------------------------------------------------------------------------------------------- |
| Один общий `styles.css`                 | Стили разделены по компонентам (`Toolbar.css`, `SlidesPanel.css`, `Workspace.css`) + глобальные |
| Выделение слайда/элемента отсутствовало | Добавлен класс `.selected` для выбранных объектов                                               |
| Цветовая схема                          | Сохранена та же, что и в ЛР-3 (визуально совпадает)                                             |

### 4. Структура файлов проекта

```
src/
 ├── common/
 │    ├── Toolbar.tsx        Панель инструментов
 │    ├── Toolbar.css
 │    ├── SlidesPanel.tsx    Левая панель со слайдами
 │    ├── SlidesPanel.css
 │    ├── Workspace.tsx      Центральная рабочая область
 │    ├── Workspace.css
 │
 ├── store/
 │    ├── editor.ts          Логика работы с презентацией (класс Editor)
 │    ├── types/             Типы (Presentation, Slide, Element)
 │    ├── functions/         Функции работы с данными
 │    ├── templates/         Шаблоны (слайды, элементы, фон и т.д.)
 │
 ├── view/
 │    └── styles.css         Глобальные стили (фон, контейнер, layout)
 │
 ├── App.tsx                 Основной файл приложения (сборка компонентов)
 └── index.tsx               Точка входа
```
