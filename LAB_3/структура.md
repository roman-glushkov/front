Структура типов

1. Presentation (Презентация)
   title — название презентации (string)
   slides — массив слайдов (Slide[])
   currentSlideId — ID текущего слайда (string)
   selectedSlideIds — массив ID выделенных слайдов (string[])

2. Slide (Слайд)
   id — уникальный идентификатор (string)
   background — фон (Background)
   elements — массив элементов на слайде (SlideElement[])

3. SlideElement (Элемент слайда)
   Может быть:
   TextElement — текстовый блок
   ImageElement — изображение

4. Background (Фон)
   {type: "color", value: string} — цветной фон
   {type: "image", value: string} — фоновое изображение
   {type: "none"} — без фона

5. TextElement (Текстовый элемент)
   type: "text" — тип элемента
   id — уникальный ID (string)
   content — текст (string)
   position — позиция ({x: number, y: number})
   size — размер ({width: number, height: number})
   font — шрифт (string)
   fontSize — размер шрифта (number)
   color — цвет текста (string)

6. ImageElement (Элемент изображения)
   type: "image" — тип элемента
   id — уникальный ID (string)
   src — путь к изображению (string)
   position — позиция ({x: number, y: number})
   size — размер ({width: number, height: number})

Функции

1. Управление презентацией
   changeTitle(presentation: Presentation, newTitle: string) — изменить название презентации
   addSlide(presentation: Presentation, newSlide: Slide) — добавить новый слайд
   removeSlide(presentation: Presentation, slideId: string) — удалить слайд по ID
   moveSlide(presentation: Presentation, slideId: string, newIndex: number) — переместить слайд на новую позицию

2. Управление элементами слайда
   addText(slide: Slide, textElement: TextElement) — добавить текстовый элемент
   addImage(slide: Slide, imageElement: ImageElement) — добавить изображение
   removeElement(slide: Slide, elementId: string) — удалить элемент по ID
   changeElementPosition(slide: Slide, elementId: string, newPosition: {x: number, y: number}) — изменить позицию элемента
   changeElementSize(slide: Slide, elementId: string, newSize: {width: number, height: number}) — изменить размер элемента

3. Работа с текстом
   changeText(slide: Slide, elementId: string, newContent: string) — изменить содержимое текста
   changeTextSize(slide: Slide, elementId: string, newSize: number) — изменить размер шрифта
   changeTextFont(slide: Slide, elementId: string, newFont: string) — изменить шрифт

4. Фон слайда
   changeBackground(slide: Slide, newBackground: Background) — изменить фон (цвет / изображение)
