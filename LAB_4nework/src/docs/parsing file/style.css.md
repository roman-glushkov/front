# Разбор styles.css

Свойства CSS и их назначение

padding — внутренние отступы элемента  
font-family — шрифт текста  
background — фон элемента  
background: linear-gradient(...) — градиентный фон  
min-height — минимальная высота элемента  
display — способ отображения (flex, block и т.д.)  
flex-direction — направление элементов в flex-контейнере  
gap — расстояние между элементами flex-контейнера  
border — рамка вокруг элемента  
border-radius — скругление углов  
box-shadow — тень вокруг элемента  
color — цвет текста  
font-size — размер шрифта  
margin-bottom / margin-top — отступы снизу/сверху  
width / height — размеры элемента  
overflow / overflow-y — как обрезать содержимое, если не помещается  
cursor — вид курсора при наведении  
transition — анимация при изменении свойства  
position — позиционирование элемента (relative / absolute)  
aspect-ratio — соотношение сторон элемента  
align-items / justify-content — выравнивание элементов в flex-контейнере  
text-align — выравнивание текста  
text-overflow — обрезка текста с многоточием  
max-width — максимальная ширина элемента  
border-color — цвет рамки  
box-shadow: 0 ... rgb(...) / ...% — тень с прозрачностью
