# Разбор main.tsx

Это точка входа React-приложения. Код берёт корневой DOM-элемент (#root) и монтирует в него компонент <App />. Используется новый API React 18 (createRoot), а StrictMode включает дополнительные проверки во время разработки.

Команды и конструкции

import — берёт экспорт из другого модуля  
React — локальная переменная, получает default export из react  
ReactDOM — локальная переменная, получает default export из react-dom/client  
App — локальная переменная, получает default export из ./App

document.getElementById('root')!
— ищет элемент в index.html с id="root".
(восклицательный знак ! — говорит TypeScript: «я уверен, что элемент точно есть»).

ReactDOM.createRoot(...).render(...)
— создаёт «корень» React-приложения (точку входа).
— рендерит (вставляет) React-компоненты внутрь найденного div#root.

<React.StrictMode>
— «режим строгой проверки»:
помогает найти ошибки, предупреждает о старом коде.
(в продакшене не влияет, только для разработки).

<App />
— главный компонент приложения,
— с него начинается всё отображение интерфейса.
